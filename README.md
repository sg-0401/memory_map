The Intelligent Photo Organization & Photobook Generation System is an AI-powered application that automatically organizes uploaded images into meaningful albums and generates professional-quality photobooks.

The system uses metadata analysis and Machine Learning-based image clustering to group similar photos, reducing the manual effort required to organize large photo collections. Users can further customize album layouts using an intuitive drag-and-drop editor before exporting the final photobook as an A4 PDF.

Features
 Automatic image organization into structured albums
 Metadata-based image grouping
 Date
 Time
 Location (if available)
 Machine Learning-based image clustering using visual features
 Secure image upload and storage using Cloudinary
 Customizable photobook layouts
 Drag-and-drop photo arrangement
 Export photobook as high-quality A4 PDF
 User-friendly React Native interface
